Where to find new features of Asp.Net Core 7


The improvements from version 6 to 7 are very minor and are only in "Minimal API".

Of course, version 7 has new improvements in Blazor, which is out-of-context for this course.


So as far as this course is concerned:

  • We are using Asp.Net Core 6 (LTS - Long Term Support) version for the first 25 sections of this course. ALL of those lectures are exactly same in Asp.Net Core 7 as well. The same code works in version 7, without NO changes.

  • The "Minimal API" (Section 26) uses Asp.Net Core 7.
